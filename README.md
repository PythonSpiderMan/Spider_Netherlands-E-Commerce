# Spider_Netherlands-E-Commerce
能解析Json就不要用Selenium！！！！

# How to run it
* Clone this repo
* replace the input.csv file with your postcode file
* cd kvk_spider
* scrapy crawl kvk_stage_1
* scrapy crawl kvk_stage_2
* scrapy crawl kvk_export
